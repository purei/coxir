    Link to this function edit(map, params) View Source 
    edit(role(), Enum.t()) :: map()

    Modifies a given role.

    Returns a role object upon success or a map containing error information.

    Params

    Must be an enumerable with the fields listed below.

    • name - name of the role
    • color - RGB color value
    • permissions - bitwise of the permissions
    • hoist - whether the role should be displayed separately
    • mentionable - whether the role should be mentionable

    Refer to this for a broader explanation on the fields and their defaults.
